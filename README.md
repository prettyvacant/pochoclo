# 🌟 Tutuca

**A personal daily life organizer — simple, warm, and built to last.**  
Track your routines, finances, weekly meals, projects and more. Designed to be used in just a few minutes a day, from any device.

→ **[Open the app](https://prettyvacant.github.io/tutuca)**

---

## What is this?

Tutuca is a Progressive Web App (PWA) — no installation needed, runs in the browser, and can be added to your phone's home screen like a native app.

Data is saved locally on your device and synced to the cloud via Supabase, so everything stays up to date across all your devices.

## Sections

| Section | What it does |
|---|---|
| 📋 **Diario** | Today's tasks, habits with checkboxes, water / sleep / mood tracker |
| 🍽 **Menú** | Weekly meal planner — vegetarian + meat option |
| 💰 **Finanzas** | Income, fixed and variable expenses, 50/30/20 rule, savings tracker |
| ✨ **Proyectos** | Project tracking with status and notes |
| 🛒 **Compras** | Shopping list — auto-generated from the weekly menu |
| 💭 **Notas** | Quick capture for ideas and reminders |

## Install as an app

**iPhone / iPad:**
1. Open the URL in Safari
2. Tap the share button ↑
3. Choose "Add to Home Screen"

**Android:**
1. Open the URL in Chrome
2. Tap the menu ⋮
3. Choose "Add to Home Screen"

---

## First time setup

When you first open the app, a setup screen lets you choose:
- Your name
- An icon (emoji) that becomes your app icon
- The interface color
- Your email for cross-device sync

All of this can be changed later from **My Profile** in the sidebar.

---

## Sync

Data syncs automatically with Supabase. To use the same data across multiple devices, enter the same email when setting up your profile on each one.

The ☁️ icon in the sidebar shows sync status. Tap **↻ Sync** to force a manual update.

## Tech

- Vanilla HTML + CSS + JavaScript (no frameworks)
- [Supabase](https://supabase.com) for cloud sync
- GitHub Pages for hosting
- Service Worker for offline support and automatic updates
